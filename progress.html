CSS- Cascading style Sheets

There can be 3 ways of implementing : Inline, Internal, External. I put the background colour to the entire file but Inline.

There are 5 types of selector :

  • Element Selector = element{...}
  • Class Selector = .class{...}
  • ID Selector = #id{...}
  • Attribute Selector = element[attribute="true"/"false"]{...}
  • Universal Selector = *{...}
